# School-Enrollment-System
A high-performance School Enrollment System developed in C. Utilizes custom data structures to implement an optimized student service queue, ensuring efficient registration flow and data processing.


The School Enrollment System is a backend application developed in C, designed to prioritize performance and algorithmic efficiency. The system manages student registration through an optimized service queue (Fila de atendimento), ensuring a fair and structured flow of incoming applications.

At its core, the queue is implemented using a Circular Queue structure (Array-based), leveraging the FIFO (First-In, First-Out) principle. This approach provides efficient, constant-time complexity for basic operations like enqueue and dequeue. The implementation focuses on memory management, pointer manipulation, and minimizing processing overhead, demonstrating a strong foundation in computer science principles and algorithmic design."
